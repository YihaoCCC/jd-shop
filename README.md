## 前言

本系统基于vue2开发。后端CMS基于 Vue3，使用 vue-cli4.x 脚手架，支持全局改色，注册登录、局部刷新、多级路由模式，无限滚动、loading优化、等目前前端主流系统架构。分离颜色样式，支持手动切换主题色，而且很方便使用自定义主题色。该商城完全个人独立开发，每一个圆角、颜色、动画，每一处布局都是作者手撸的。项目供学习使用。
